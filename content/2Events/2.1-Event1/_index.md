## Event Objectives

The event was organized to create a dedicated space for networking and direct learning within the tech community. Its core objective was to deliver the most practical experiences, insights, and real-world perspectives from industry experts (speakers) currently operating across diverse corporate environments. Consequently, this session aimed to help attendees grasp the realities of large-scale system operations and current technological trends.

## Main Content of the Event

The sharing session dove deep into Cloud Infrastructure (particularly on the AWS platform) and the shifting landscape of technical roles under the influence of AI. Key focus areas included:

- The future of Developer, DevOps, and Cloud Engineering professions as AI tools advance rapidly.
- Resolving architectural complexity as enterprises scale large systems and migrate cloud infrastructure.
- Leveraging AI for cloud cost optimization (FinOps) and automated security assessments (Security, Pen-testing).
- Analyzing real-world architectural scenarios and cost estimation when utilizing AWS services such as Route 53 Resolver, EC2, ALB, or AWS Secrets Manager.

## Key Highlights

### 1. AI Cannot Fully Replace Humans in Cloud Engineering
Despite AI's rapid code generation and deployment speeds, enterprises still inherently rely on skilled engineers to handle live incidents and make accurate, high-stakes decisions when managing critical infrastructure.

### 2. The Competitive Advantage of AI in FinOps
Leveraging AI to optimize cloud spending yields superior efficiency. An AI assistant that understands both AWS architectural topology and financial logic can manage budgets far better than a traditional financial accountant who lacks cloud infrastructure insights.

### 3. The Reality of "Hidden" Fees on the Cloud
By dissecting actual project expenditures, the event highlighted the reality that data transfer costs are sometimes negligible compared to the baseline operational fees of foundational cloud services (for instance, setting up a private network costs at least 180 USD/month just for Route 53).

## Knowledge and Experience Gained

Hearing experts directly dissect AWS architectures brings a high level of practicality and depth to full-stack web development. Instead of simply getting bogged down in writing backend code or designing SQL Server databases to run smoothly on a local environment, the insights from this event demonstrate that when deploying a project to production, optimizing infrastructure costs and establishing robust security are the ultimate survival factors. Integrating AI into operational workflows also opens up a new paradigm: rather than focusing strictly on logical code design, data structures, and algorithms, I must also learn to leverage automated tools to evaluate system vulnerabilities and manage infrastructure efficiently.

## Evaluation and Reflections

The event provided an immensely valuable and practical experience, especially for a student. While university lectures lay down a solid theoretical foundation, the real-world battle stories from FCAJ exposed the sheer complexity involved when a system scales up to serve an actual enterprise. Looking at the highly granular cost estimations for each service, my initial sense of overwhelm quickly turned into genuine excitement. This session motivates me to dive deeper into cloud architecture, ensuring that I can not only build smooth-running mobile applications or e-commerce platforms but also design flexible, cost-optimized infrastructure ready to handle heavy production traffic.
