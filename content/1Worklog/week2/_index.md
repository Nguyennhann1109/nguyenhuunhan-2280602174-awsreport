## Objectives

- Learn the core services of Amazon Web Services (AWS).
- Understand the roles of Compute, Database, Security, and Container services in AWS architectures.
- Build a solid foundation for the upcoming hands-on labs and workshops.

After completing the second week, I achieved the following outcomes:

- Understood the purpose and functionality of the core AWS services.

- Became familiar with access control using AWS Identity and Access Management (IAM).

- Built a strong technical foundation for the upcoming labs and workshop activities.
